# Get-SPOBuiltInFontPackageSettings

## SYNOPSIS

Use this cmdlet to get settings of [built-in font packages](/sharepoint/brand-center-font-packages).
 
## SYNTAX

```powershell
Get-SPOBuiltInFontPackageSettings
```

## DESCRIPTION

Use this cmdlet to get settings of [built-in font packages](/sharepoint/brand-center-font-packages).

## EXAMPLES

### Example 1

```powershell
Get-SPOBuiltInFontPackageSettings
```

This example retrieves the current settings of built-in font packages.
